Introduction

Bronson is an opinionated module application framework used for building large scalable web applications. Bronson sits on top of RequireJS and JQuery offering libraries for the loading, unloading, messaging and bootstrapping of modules. The idea being that it's impossible to build large manageable JavaScript applications and you are better served in building smaller modular components that when combined form your larger web application.

Features

  • Module loading & unloading through Bronson.Core
  • Univeral interface offered through Bronson.Api
  • Managed publish & subscribing of events through Bronson.Core & Bronson.Permissions

Dependencies

  • jQuery
  • RequireJS
