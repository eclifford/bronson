What is Bronson?

Bronson is an opinionated module application framework for the building of large scalable web applications. It was born from the need to modular application development in the real world not just single page app prototypes. Bronson sits on top of RequireJS to give a common API for loading modular widgets that you can control programatically with a universal interface for starting, stopping, loading, unloading, and communicating of modules on your pages.

Features

  • Module boilerplate reflecting the typical life cycle of a standalone widget.
  • API for loading, unloading, starting, stopping modules
  • Fast, managed, framework agnostic publish/subscribe system built for module to module as well as application wide communication.
  • Permissions controller for management of communication
  • Cross browser tested IE8 + Modern Browsers.
  • Only dependency is RequireJS

Installation

Bronson is can be loaded via script tag or through require

Dependencies

  • RequireJS
