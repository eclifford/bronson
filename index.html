Introduction

Bronson is a collection of utilities for building large, scalable modular javascript applications.

Features

  • Managed Publish/Subscribe
  • Module loader
  • Base Backbone Model, View, Collection
  • Optimized Memory Management

Depedencies

  • RequireJS
  • Jquery
  • Underscore
  • Backbone

    • Overview

    Api

    The API layer is the public interface for communicating with the Bronson library. The API abstracts the Core and Permissions layer and presents a closed to modification interface. Loading a Module 

    # With Library already loaded
Bronson.Api.createModule 'TestModule', 
  url: 'path/to/data'
, ->
  console.log 'module created'

# VIA AMD/RequireJS       
require ['bronson'], (Bronson) ->            
  Bronson.Api.createModule 'app/modules/model-listing',   
    url: '/app/data/model-list-data.json'          
  , ->       
    console.log 'module created'
    Publishing an Event 
    require ['bronson'], (Bronson) ->                     
  Bronson.Api.publish 'TestEvent', { foo: "bar" }
    Subscribing to an Event 
    require ['api'], (api) ->                     
  api.createModule 'app/modules/model-listing',   
    url: '/app/data/model-list-data.json'          
  , ->       
    console.log 'module created'
